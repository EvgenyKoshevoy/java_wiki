Объекты типа String

    Конструкторы

    String(char[] chars) - строка инициализируемая массивом сомволов
   
    String(char[] chars, int начальный_индекс, int количество_символов) - для инициализации указывается определенная часть массива

    Для byte[] конструкторы ананлогичны. Используется для работы с символами в кодировке ASCII

    String(String строковый_объект) - создает строку на основе другой строки

    String(StringBuffer объект_буфера_строк) - создает строку из объекта StringBuffer

    String(StringBuilder объект_построения_строки) - создает строку из объекта StringBuilder

   
       

Методы

    Длина символьной строки:
        int length() - возвращает длину строки

    Сцепление символьных строк с другими типами данных:
        int age = 9;
        String s = "Ему " + age + " лет";

    Преобразрвание символьных строк:
        String toString() - для удобочитаемого отображения объектов

    Извлечение символов:
        char charAt (int index) - возаращает символ, по указанному индексу

        void getChars(int начало_источника, int конец_источника, char[] target, int начало_индекса)
            начало_источника - индекс начала подстроки
            конец_источника - индекс конца извлекаемой подстроки (в извлекаемою подстроку не входит)
            target - массив, принимающий символы
            начало_адресата - с какого индекса

        byte[] getBytes - применяется там, где не поддерживаются 16 разрядные символы в Юникоде (используется для кодировки ASCII)

        char[] toCharArray() - преобразует строку в новый массив символов (аналог getChars())

    Сравнение символьных строк:
        boolean equals(Object строка) - сравнение символьных строк на равенство. Возвращает true если строки лексически равны. (Оператор == сравнивает ссылки на объекты).

        boolean equalsIgnoreCase(Object строка) - сравнение строк без учета регистров

        boolean regionMatches(int начальный индекс, String строка2, int индекс начала строки, int количество_символов) - сравнивает одну часть смивольной строки с другой частью

        boolean startWith(String строка) - определяет начинается ли заданный объект типа String с указанноый символьной строки
       
        boolean endWith(String строка) - определяет завершается ли заданный объект типа String указанной подстрокой
        Пример, заканивается ли файл ".txt":
            String fileName;
                if(!fileName.endsWith(".txt")) {
                    //...
                }
 
        int compareTo(Object o) - сравнивает данную строку с другим объектом по длине. Строка меньше другой, если она следует перед ней в лексикографическом порядке. Если меньше - возвращает: -1, равна: 0, больше: 1.

    Поиск в символьных строках:
        int indexOf(char символ) - поиск первого вхождения символа в строке. Возвращает позицию символа в строке. При неудачном поиске возвращает -1.

        int lastIndexOf(char символ) - поиск последнего вхождения символа в подстроке. Возвращает последнюю позицию символа в строке. При неудачном поиске возвращает                                                                   -1.

        int indexOf (String str) - поиcк подстроки, определяемый параметром str. Возвращает индекс первого вхождения подстроки str или -1 если описк завершен неудачно

        int last indexOf (String str) - поиcк подстроки, определяемый параметром str. Возвращает индекс последнего вхождения подстроки str или -1 если описк завершен неудачно.

    Видоизменение символьных строк:
        String substring(int начальный_индекс) - извлекает подстроку из указанной строки с начального_индекса и до конца.
        String substring(int начальный_индекс, int конечный_индекс) - то же самое, но указывается с какого индекса по какой (конечный индекс - 1).

        String concat(String строка) - объединяет вторую строку с первой.

        String replace(char исходный, char заменяемый) - заменяет все исходные символы на заменяемый.
        String replace(CharSequence исходная, CharSequence заменяемая) - одна последовательность символов заменяется другой.

        String trim() - возвращает копию символьной строки из которой удалены все начальные и конечные пробелы.

    Преобразование данных методом valueOf():
            static String valueOf() - предоставляет строковое представление переданных примитивных типов.
                Метод перегружен для всех типов данных Java.
            static String valueOf(char[] chars, int начальный_индекс, int количество_символов) - строковое представление массива символов.

    Смена регистра букв в строке:
        String toLowerCaes() - преобразует все символы в данной строке в ижний регистр

        String toUpperCase() - преобразует все символы в строке в верхний регистр

    boolean contentEquals(StringBuffer sb) - возвращает true если строка представляет собой ту же последовательность символов как указано в буфере строки

    static String copyValueOf(char[] data) - возвращает строку, которая представляет собой последовательность символов в указанный массив

    contains("value") - проверка строки на содержание заданного значения value

    String[] split(String regex) - разделяет строку вокруг данного регулярного выражения. Возвращает массив строк, разделенных вокруг данного регулярного выражения.
        Пример:
            file.split(" "); - разделит строку, вернет массив строк.
    Проверка строки на пустоту:
        string.trim().length() == 0, где string - имя переменной типа String

      Класс StringBuffer

    В отличии от String класс представляет собойрасширяемые и доступные для изменения последовательности символов.

    Конструкторы:
        StringBuffer()
        StringBuffer(int размер)
        StringBuffer(String строка)
        StringBuffer(CharSequence символы)

    Методы:

        length() - получить текущую длину

        capacity() - выводит текущий объем выделенной памяти

        void ensureCapacity(int минимальная_емкость) - предварительно выделить место для определенного количества символов после создания объекта

        void setLength(int длина) - задать длину символьной строки

        char charAt(int где) - извлечь символ

        char setCharAt(int где, char символ) - добавить символ по указанному индексу

        void getChars(int начало_источника, int конец_источника, char target[], int начало адресата) - скопировать подстроку в массив

        StringBuffer append(String строка) - присоединяет строковое представление любого другого типа
                                         (int число)
                                         (Object объект)

        StringBuffer insert(int индекс, String строка/char символ/Object объект) - вставляет одну символьную строку в другую

        StringBuffer reverse() - изменяет порядок следования символов

        StringBuffer delete(int начальный индекс, int конечный_индекс) - удаляет последовательность символов из вызывающего объекта

        StringBuffer deleteCharAt(int позиция) - удалить смивол на указанной позиции

        StringBuffer replace(int начальный_индекс, int конечный_индекс, String строка) - заменить один набор символов другим

        StringBuffer substring(int начальный_индекс) - получить часть содержимого объекта StringBuffer
                                            (int начальный_индекс, int конечный_индекс)

    Класс StringBuilder

    Отличие от StringBuffer в том, что он не является потокобезопасным. Дает выйгрыш в производительности за счет этого.
