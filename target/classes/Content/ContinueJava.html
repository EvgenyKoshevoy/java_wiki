Оператор continue в Java

Оператор continue - это один из операторов перехода. Он игнорирует часть кода.

Continue позволяет опустить участок кода и вернуться в начало цикла.

Синтаксис

continue;

В коде будет выглядеть так:

     class Test {
 
         public static void main (String [] args) {
 
             for (int i = 1; i <= 7; i++){
                 System.out.println(i + ": я выполянюсь перед continue!");
                 if (i > 2)
                     continue;
                 System.out.println("     "+ i + ": я выполянюсь всего два раза ;(");
             }
         }
     }

Перед нами цикл for. Допустим нам нужно чтобы первая строка выполнялась 7 раз:
System.out.println(i + ": я выполянюсь перед continue!");

А вторая строка всего два раза:
System.out.println(" "+ i + ": я выполянюсь всего два раза ;(");

Перед вторым выводом в консоль задаем условие: 
     if (i > 2);
	 continue;

Таким образом когда i будет больше 2 оператор continue будет перенаправлять в начало цикла, вторая строка не будет выведена в консоль. Все, что после continue - игнорируется.
