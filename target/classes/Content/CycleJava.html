Циклы в Java

В Java выделяют 4 виды циклов:

1. Цикл While

2. Цикл do...while

3. Цикл for

4. Цикл for each

Цикл While

Цикл while еще называют цикл "пока" или "до тех пор пока"

Общая форма записи цикла while выглядит так:

while (условие) { // действия, которые выполняются пока условие истинно; }

Суть оператора while: он повторяетя действия(-я) до тех пор, пока условие истинно.

Например:

     до тех пор пока (в кошельке > = 200 грн) {
     вожу девушку по кафешкам;
     покупаю девушке цветы;
     }

В приведенном примере:

1. "до тех пор пока" - это while

2. (в кошельке > = 200 грн) - это условие

3. "Вожу девушку по кафешкам", "Покупаю цветы" - это действия, которые будут выполняться пока условие истинно.

Задача №1

Необходимо вывести на экран числа от 1 до 7.

Решение:

     class Test{
         public static void main(String args[]){
             int i = 1;
             while (i < 8) {
                 System.out.print(i + " ");
                 i++;
                 }
         }
     }

Задача №2

Если бы код был написан так, то он не выполнился бы ни разу.

     class Test {
         public static void main(String args[]) {
             int i = 1;
             while (i < 0) {
                 System.out.print(i + " ");
                 i++;
             }
         }
     }

Не выполнился бы потому, что int i = 1; а в условии указано while (i < 0). В коде i уже равно 1, что не соответствует условию.

Задача №3

Если бы код был написан так, то на экран выводились бы цифры от 1 до бесконечности.

     class Test {
         public static void main(String [args]) {
             int i = 1;
             while (true) {
                 System.out.print(i + " ");
                 i++;
             }
         }
     }

Это пример бесконечного цикла. При запуске он заблокировал бы программу т.к. код не останавливается потому что условие true и есть истинное условие.

Цикл do...while

  • Цикл do...while еще называют "делать до тех пор пока"
  • Цикл do...while - это одна из вариаций цикла while

Общая форма цикла do...while:

do {
// действия, которые выполняются
}
while (условие);

Отличия этих циклов рассмотрим в примерах:

Задача №5 - код с циклом while

     class Example1 {
         public static void main(String args[]) {
             int i = 1;
             while (i < 0) {
                 i++;
                 System.out.print(i + " ");
             }
         }
     }

Комментарий:

Данный код не выполнится ни разу, т.к. сначала идет проверка условия while (i < 0) и только потом, если условие истинно выполняется код. При цикле while идет предпроверка условия.

Задача №6 - код с циклом do...while

     class Example2 {
         public static void main(String args[]){
             int i = 1;
             do {
                 i++;
                 System.out.print(i + " ");
             } while(i < 0);
         }
     }

Комментарий:

do...while сначала выполняет тело цикла, а потом проверяет условие, поэтому тело цикла как минимум выполнится 1 раз. При цикле do...while происходит постпроверка условия.

Цикл do...while всегда выполняется как минимум 1 раз!

Цикл do...while удобно использовать, когда некоторое действие в программе нужно выполнить по крайней мере единожды, но при некоторых условиях его придется повторять многократно.

Цикл for

Цикл for еще называют "цикл со счетчиком"

Когда заранее известно сколько раз повториться одно и то же действие - необходимо использовать цикл for.

Общая форма записи:

Общая форма цикла for в Java

     for (инициализация; условие; итерация) {
	 
     // действия, которые выполняются 
	 
     }

Пример:

     class Test {
         public static void main(String args[]){
             for (int i = 1; i <= 5; i++) {
                 System.out.println (“Супер!”);
             }
         }
     }

Комментарий:

Конструкция for (i = 1; i <= 5; i++) в языке Java - это все равно, что сказать "Повторить столько то раз".

int i = 1; - это первый параметр, который отвечает за установку начального значения. Этот параметр называется инициализация, где i - переменная цикла.

i <= 5; - это второй параметр, который отвечает за условие, при котором будет выполняться цикл.

i++ это третий параметр, который отвечает за изменение переменной i после каждого шага цикла. Этот параметр называется итерация.

Решение:

	         }

     }
