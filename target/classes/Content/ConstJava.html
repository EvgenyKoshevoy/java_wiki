Константы в Java

Как сделать переменную константой?

1. Чтобы сделать переменную константой необходимо использовать ключевое слово final

.

2. Константы принято писать большими буквами, чтобы отличить их от обычных переменных.

3. Если в программе происходит попытка изменить константу, компилятор javac выдаст ошибку, потому что значение константы можно задать 1 раз.

Задача:

Вычислить периметр круга при разных значениях радиуса:

R = 10 см

R = 25 см

Решение:

Периметр круга вычисляется по формуле P = 2πR

P - периметр круга (длина окружности)

π - Число Пи (3,1415)

R - радиус круга (окружности)

	     class Test {
 
             public static void main(String[] args) {
                 final double Pi = 3.1415926536;
                 // Константа Pi с типом данных для чисел c плавающей точкой
                 double lengthl = 2*Pi*10;
                 double length2 = 2*Pi*25;
                 System.out.println ("Периметр круга при радиусе 10 см равен " + lengthl + ", а при радиусе 25 см равен " + length2);
             }
         }

Комментарии к коду:

final double Pi = 3.1415926536;
В этой строчке переменную с названием Pi сделали константой, добавив в начале ключевое слово final. Если изменить значение этой величины, компилятор javac выдаст ошибку, т.к. значение переменной можно задавать только 1 раз.
