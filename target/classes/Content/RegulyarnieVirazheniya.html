Регулярные выражения

    Обработка регулярных выражений поддерживается классами Pattern и Matcher.

    Класс Pattern

    Конструкторы не определяются. Вместо этого вызывается фабричный метод compile():
        static Patter compile(String шаблон)
            шаблон - регулярное выражение, которое требуется использовать.

    Метод преобразует шаблон в символьную строку для сопоставления средствами класса Matcher. Возвращает объект типа Pattern.

    Как только объект Pattern получен, можно вызвать фабричный метод matcher() для создания объекта Matcher:
        Matcher matcher(charSequence строка)
            строка - последовательность символов сопоставляемая с шаблоном, называется входной последовательностью.

    Так же методу можно передать символьную строку.

    Класс Matcher

    Не имеет конструкторов. Для создания используется фабричный метод matcher(), описанный в классе Pattern.

    Методы:

        boolean matches() - определяет совпадает ли последовательность символов с шаблоном.
            Возварщает true если совпадает вся последовательность (для проверки совпадения подпоследовательности используется другой метод),                         иначе false.

        boolean find() - определяет совпадает ли подпоследовательность с шаблоном.
            Возвращает true, если подпоследовательность совпадает с шаблоном, иначе false. Можно вызвать неоднократно, при каждом вызове сравнение начинается с того места, где было завершено предыдущее сравнение.

        String group() - возвращает символьную строку, содержащую последнюю совпавшую последовательность. Если совпадений не обнаружено генерируется исключение IllegalStateException.

        int start() - возвращает индекс текущего совпадения во входной последовательности символов.

        int end() - возвращает индекс следующий, после текущего совпадения.
       
        String replaceAll(String новая_строка) - каждую совпавшую последовательность заменяет другой последовательность новая_строка.
