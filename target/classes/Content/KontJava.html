Конкатенация строка в Java

Конкатенация - это соединение двух строк в одну.

Пример 1:

public class Test{

public static void main(String[] args) {
String morning1 ="доброе";
String morning2 ="не доброе, потому что сегодня понедельник";

System.out.println("Сегодня утро " + morning1);
System.out.println("Сегодня утро " + morning2);
}
}

Если запустить код то получится:
Сегодня утро доброе
Сегодня утро не доброе, потому что сегодня понедельник

Пример 2:

public class Test {

public static void main(String[] args) {

String name = "Виктор";
String car = "Porsche 918 Spyder";
System.out.println(name + " купил " + car);
}
}

Результат программы:
Виктор купил Porsche 918 Spyder

В данных примерах конкатенация осуществлена с помощью оператора "+".
Конкатенацию можно осуществлять не только с помощью оператора +, есть еще несколько способов. Часто используемый .concat()
