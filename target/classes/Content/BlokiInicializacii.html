Блоки инициализации

    Инициализация - когда впервые задаем занчение переменной.

    Блоки инициализации используются для задания начальных значений переменным.

    Виды блоков инициализации:
    - нестатические;
    - статические;

   Нестатические блоки

    Используется для инициализации любых типов переменных, кроме статических.
class Dog{
 
    private String name;
    private String poroda;
    private int age;
 
    { //блок инициализации
        name = "Шарик";
        poroda = "овчарка";
        age = 2;
    }
 
    public Dog(String x, String y, int z){
        name = x;
        poroda = y;
        age = z;
    }
}
   
    Также внутри блокоа инициализации можно писать любые команды (использовать как метод).
class Dog{
 
    private String name;
    private String poroda;
    private int age;
 
    {
        System.out.println("Это нестатический блок инициализации!");
    }
    
    public Dog(String x, String y, int z){
        name = x;
        poroda = y;
        age = z;
    }
 
}
public class Test {
 
    public static void main(String[] args) {
        Dog d = new Dog("Шарик", "овчарка", 2);
    }
}
    Вывод:
   
    Это не статический блок инициализации!

    Статические блоки

    Используются для инициализации статических переменных. Код статического блока выполняется при первой загрузке класса.

 
public class Solution {
    public static Map<Double, String> labels = new HashMap<Double, String>();

    static {
        labels.put(0.1, "One");
        labels.put(0.2, "two");
        labels.put(0.3, "three");
        labels.put(0.4, "for");
        labels.put(0.5, "five");
    }

    public static void main(String[] args) {
        System.out.println(labels);
    }
}
