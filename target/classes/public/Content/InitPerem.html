Инициализация переменной

Инициализация - присвоение начального значения переменной.

Если переменная создана и ей не присвоено ни одно значение, то она не инициализирована.

Для переменных типа float к значению обязательно добавляем букву f. Пример: float pi = 3.14f;

При присвоении переменной типа String, значение заключает в двойные кавычки. Пример: String title = "Обожаю Java";

При присовении переменной типа char, значение заключаем в одинарные ковычки. Пример: char letter = 'M';
