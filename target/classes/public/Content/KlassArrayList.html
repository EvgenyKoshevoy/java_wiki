Класс ArrayList

Синтаксис:
    ArrayList <тип_переменной> name = new ArrayList();

    Размер такого массива расширяется автоматически. При заполнении он создает новый массив равный длине (длина_старого * 1,5) + 1. Данные копируются в новый, а старый удаляется сборщиком мусора.

add() - добавление нового элемента в массив.

    ArrayList<Cat> cats = new ArrayList();
    cats.add(new Cat("Бегемот"));

Новый элемент добавляется в конец списка.

indexOf() - поиск индекса объекта по ссылке на объект.

    ArrayList<Cat> cats = new ArrayList();

    Cat thomas = new Cat("Thomas");
    Cat behemoth = new Cat("Behemoth");

    cats.add(thomas);
    cats.add(behemoth);
   
    int thomasIndex = cats.indexOf(thomas); //значение = 0

get() - поиск элемента по индексу

    Cat secondCat = cats.get(1);

contains() - проверка содержит ли массив конкретный объект

    cats.contains(thomas); //результатом является true или false

set() - в отличии от add() затирает тукещуу значение и заменяет его новым

ArrayList позволяет вставлять элементы в конкретную ячейку:

    add(int index, Cat elenemt); //сначала сдвигает все эелементы начиная с [index] а в образовавшееся пустое место вставляет значение
    set(int index, Cat element); //затирает старое значение, хранящееся в ячейке

    cats.set(0, philip);
    cats.add(0, philip);

clear() - полностью очищает список

toString() - выводит список в формате строки
   
    System.out.println(cats.toString());

size() - возвращает число элементов

remove() - удаление элемента из массива

Может удалять не только по индексу, но и по ссылке на объект.

Пример: cats.remove(philipp);

Нельзя одновременно проводить итерацию коллекции и ее изменение. Может возникнуть ошибка  выхода за пределы списка. Для корректного удаления используется класс Iterator.

Класс Iterator отвечает за безопасный проход по списку элементов. Имеет всего три метода:

hasNext() - возвращает true или false в зависимости от того есть ли в списке следующий элемент или мы дошли до последнего.

next() - возвращает следующий элемент списка

remove() - удаляет элемент из списка

Пример:

Iterator<Cat> catIterator = cats.iterator(); //создаем итератор

while(cat.Iterator.hasNext()) { //до тех пор, пока в списке есть элементы

    Cat nextCat = catIterator.next(); //получаем следующий элемент
    if (nextCat.name.equals("Филипп Маркович")) {
        catIterator.remove(); //удаляем кота с нужным именем
    }   

}
