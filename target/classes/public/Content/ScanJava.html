Работа со сканером в Java

Сканер предназначен для считывания с консоли введенных данных (числа, либо слова).

Сейчас мы работаем с пакетом java.util. В нем есть класс Scanner и у него есть методы.

Методы - это действия, которые может выполнить Scanner.

Чтобы использовать сканер необходимо выполнить три шага:

Шаг №1: Сделать импорт сканера из пакета java.util
import java.util.Scanner;

Шаг №2: Объявить сканер:
Scanner scan = new Scanner(System.in);

Шаг №3: Вызвать соответствующий метод для считывания с консоли. Например:
int number = scan.nextInt();

Познакомимся с четырьмя методами сканера:
  1. next (); - считать слово
  2. nextLine (); - считать целую строку
  3. nextInt (); - считать число типа Int
  4. nextDouble (); - считать число типа double
  5. netxByte(); - считать байт
  6. nextShort(); - считать число типа short
  7. nextLong(); - считать число типа лонг
  8. nextFloat(); - считать число типа float
  9. nextBoolean(); - считать число булевого типа

Пример №1 - с методом nextInt()

Задание: ввести любое число, программы должны вывести какое число мы ввели

Решение:

		 import java.util.Scanner; // импорт сканера
         
         class Test {
	         public static void main(String args[]){
		         System.out.print("Введите любое целое число от 1 до 10: ");
		         Scanner scan = new Scanner(System.in);
		         int number = scan.nextInt();
		         System.out.println ("Вы ввели число " + number);	 	
		     }
		 }

Комментарий:

Scanner scan = new Scanner (system.in);
С помощью Scanner scan мы указали что переменная будем иметь имя scan и будет относиться к классу Scanner. Обязательно перед именем переменной прописать Scanner, а имя можно задать любое.

int number = scan.nextInt();
С помощью int number мы указали, что переменная number относится к типу int.
= scan.nextInt(); - метод сканера nextInt() отвечает за то, чтобы считать введенное пользователем число с консоли.

Пример №2 - с методом nextInt()

Задание: попросить ввести ввести два числа, после вывести на экран их сумму

Решение:

		     import java.util.Scanner; // импорт сканера
 
             public class Test {
	             public static void main(String[] args) {
 
		             Scanner k = new Scanner(System.in);
		             System.out.println ("Введите 2 любых целых числа: ");
		             int number1 = k.nextInt();//считывает первое число и присваивает значение в number1
		             int number2 = k.nextInt();//считывает второе число и присваивает значение в number2
		             System.out.print(number1 + number2);//выводит сумму number1 + number2
	             }
             }

Пример №3 - с методом nextLine()

Задание: запросить ввод слова или фразы, а затем вывести его на экран.

Решение:

		     import java.util.Scanner; // импорт сканнера
 
             public class Test {
	             public static void main(String[] args) {
 
		         Scanner sc = new Scanner(System.in);
		         System.out.println("Введите любое слово или фразу: ");
		         String phrase1 = sc.nextLine();
		         System.out.println(phrase1);
	             }
             }

Комментарий:

Шаг №1: Импортировали сканер из пакета java.util:
import java.util.Scanner; // импорт сканера

Шаг №2: объявили сканер:
Scanner sc = new Scanner(System.in);

Шаг №3: считали слово или фразу с консоли
String phrase1 = sc.nextLine();

  • С помощью String phrase1 мы объявили переменную с именем phrase1 относящуюся к типу String
  • = sc.nextLine(); отвечает за считываение введенного пользователем слова или фразы. То есть с помощью метода nextLine() считываем введенное слово или фразу и присваеваем его переменной phrase1

Пример №4 - с методом nextLine()

Задание: сделать запрос на ввод нескольких фраз и вывести их одной строчкой.

Решение:

		     import java.util.Scanner; // импорт сканера
 
             public class Test {
                 public static void main(String[] args) {
 
                 Scanner sc = new Scanner(System.in);
                 System.out.println("Введите любые 2 слова или фразу: ");
                 String phrase1 = sc.nextLine();
                 String phrase2 = sc.nextLine();
                 System.out.println(phrase1 + " " + phrase2);
                 }
             }

Пример №6 - с методом nextDouble()

Задание: сделать запрос на ввод дробного числа, вывести введенное число на консоль

Решение:

	     import java.util.Scanner; // импорт сканера
 
         class Test {
	         public static void main(String args[]){
		         System.out.print("Введите любое дробное число: ");
		         Scanner scan = new Scanner(System.in);
		         double number = scan.nextDouble();
		         System.out.println ("Вы ввели число " + number);
	        }
         }
