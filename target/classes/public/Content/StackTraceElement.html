StackTraceElement

Пример:
    StackTraceElement[] stackTraceElements = Thread.currentThread().getStackTrace();

    В Java есть специальная коллекция  - стек (stack). Когда одна функция вызывает другую, Java-машина помещает в этот стек новый элемент StackTraceElement. Когда функция завершается, этот элемент удаляется из стека.
    Каждый StackTraceElement содержит информацию о методе, который был вызван.

Что содержится в stackTraceElements[]:
    0 - getStackTrace
    1 - имя метода (свое имя)
    2 - имя метода кто вызывал [1]
    3 - имя метода кто вызывал [2]
    4 - и т.д. ...

Методы:

- getMethodName() - получает имя метода.
Пример:
    StackTraceElement[] stackTraceElements = Thread.currentThread().getStackTrace();
    String name = stackTraceElements[2].getMethodName();

- getLineNumber() - возвращает номер строки кода из которого вызвали этот метод
Пример:
          int numberLine = stackTraceElements[2].getLineNumber();

- getClassName() - возвращает имя класса
