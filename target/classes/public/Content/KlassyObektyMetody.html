Классы

class имя_класса {
    тип переменная1;
    тип переменная2;
    //...
    тип переменнаяN;

    //объявление методов
    тип метод1 (параметры) {

    }
    тип метод2 (параметры) {
        //тело метода
    }
    //...
    тип методN (параметры) {
        //тело метода
    }
}

Абстрактный класс


- Может содержать объявление метода без его реализации.

public abstract class ChessItem {
    public int x, y;
    private int value;

    public int getValue() {
        return value;
    }

    public abstract void draw();     //абстрактный метод, реализация отсутствует
}

Класс, содержащий один или несколько абстрактных методов должен быть объявлен как абстрактный.

- У него не может быть объектов.

- Подкласс, наследующий абстрактный класс должен реализовать все абстрактные методы суперкласса, иначе он так же должен быть объявлен как абстрактный.

- Атрибут abstract наследуется до тех пор, пока не будет достигнута полная реализация класса.

Использовать если:
    - хочешь поделиться кодом между несколькими тесносвязанными классами;
    - что классы, расширяющий абстрактный класс, имеют много общих методов или полей, или требуют других модификаторов доступа кроме public (protected или private);
    - нужно объявить не статические или не-final поля. Это позволяет определять методы, которые могут получить доступ и изменить состояние объекта, которому они принадлежат.

   


Вызов перегруженных конструкторов через this()


class Cat{
    int age;
    int birthday;
    
    // Инициализируем переменные явно
    Cat(int i, int j) {
        age = i;
        birthday = j;
    }
    
    // Инициализируем переменные одним и тем значением
    Cat(int i) {
        this(i, i); // вызывается Cat(i, i);
    }
    
    // Присвоим значения по умолчанию 0
    Cat() {
        this(0); // вызывается Cat(0);
    }
}

Оператор new

Общая форма:
переменная_класса = new имя_класса(список_аргументов);

Оператор new возвращает ссылку на вновь созданный объект, которую получает переменная_класса в результате присваивания.
