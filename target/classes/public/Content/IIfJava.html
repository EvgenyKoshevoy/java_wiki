Условный оператор if в Java. Оператор switch.

В Java есть так называемые конструкции ветвления:

1. Условный оператор if

2. Оператор switch

Общая форма улсловного оператор if такая:

if (условие) {
//действие(-я), которые выполняются, если услвое истинно;
}
else if (условие) {
//действие(-я), которые выполняются, если услвое истинно;
}
else if (условие){
//действие(-я), которые выполняются, если услвое истинно;
}
else (условие) {
//действие(-я), которые выполняются, если услвое истинно;
}

Важно:

  • Оператор всегда начинается со слова if, после которого идут скобки с условием.
  • После круглых скобок никогда не ставится точка с запятой.
  • Чтобы указать альтернативный вариант (если не выполняется, тогда...) используется слово else.
  • Если условий несколько, то кажде из них будет выполнятся комбинацией else if. Последний вариант записывается через else без условия.
Возможные операции сравнения:
< - меньше;
<= - меньше или равно;
> - больше;
>= - больше или равно;
== - равно;
!= - не равно.

Задача №1

Необходимо написать программму, где требуется вывести запрос на ввод числа от 1 до 3, п рограмма должна сказать, какое число ввел пользователь.

Решение:

         import java.util.Scanner; // подключаем сканер		 
         class Test {
	         public static void main (String [] args) {
 
		         System.out.print ("Введите число 1, 2 или 3: "); 
		         Scanner inputFigure = new Scanner (System.in); 
		         int i = inputFigure.nextInt (); 
		         if ( i==1) { 
			         System.out.println ("Вы ввели число 1");
		             }
		         else if (i==2) {
			         System.out.println ("Вы ввели число 2");
		             }
		         else if (i==3) {
			         System.out.println ("Вы ввели число 3");
		             }
		         else {
			         System.out.println ("Вы ввели число не равное 1, 2 или 3");
		             }
	         }
         }

Комментарии:

1. В условии сказано, что нам нужно ввести число на выбор 1, 2 или 3. Первое что нужно сделать - предлождить ввести число. Поэтому пишем в коде:

System.out.print ("Введите число 1, 2 или 3: ");

2. Пользователь вводит число, а мы должны его считать с консоли. Для этого подключаем сканнер:

import java.util.Scanner;

3. Далее с помощью сканера присваеваем в переменную i число, которое ввел пользователь:

Scanner inputFigure = new Scanner (System.in);
int i = inputFigure.nextInt ();

4. Проверяем равно ли i 1. Если i равно 1, будет выведено на экран сообщение "Вы ввели число 1"

if ( i==1) {
System.out.println ("Вы ввели число 1");
}

5. Если i не равно 1, то проверяем равно ли i 2. Если i равно 2 выводится на экран сообщение "Вы ввели число 2"

else if (i==2) { System.out.println ("Вы ввели число 2"); }

6. Если i не равно 1 или 2 проверяем равно ли i 3. Если i равно 3 выводится на экран сообщение "Вы ввели число равное 3"

else if (i==3) { System.out.println ("Вы ввели число 3"); }

7. Если все предыдщие шаги оказались false, оператор перейдет к выполнению кода, который написан в else и выведет сообщение "Вы ввели число не равно 1, 2 или 3"

else { System.out.println ("Вы ввели число не равное 1, 2 или 3"); }

И оператор будет закончен. Конец программы.

8. Обратить внимание на то, что в if (i == 1) используется двойное равеноство ==. Очень распространенная ошибка использовать = (присвоение значения переменной) вместо == (проверка равна ли переменная указанному значению).

Задача №2

Необходимо написать программу, где предлагается ввести число 1. Если введно число 1, программы должна вывести "Вы ввели число 1". Если пользователь ввел другое число, программа должна вообще ничего не делать.

Решение:

     import java.util.Scanner; // подключаем сканер
     class Test {
	     public static void main (String [] args) {
 
		     System.out.print ("Введите число 1: "); 
		     Scanner inputFigure = new Scanner (System.in); 
		     int i = inputFigure.nextInt (); 
		     if ( i==1) { 
			     System.out.println ("Вы ввели число 1");
		         }
	     }
     }

Комментарий:

Как видно из решения задачи if есть, но else нет.

Так сделано потому, что выполнять код требуется только когда условие истинно.

Оператор switch (ветвление с помощью переключателей)

Конструкции с if else при большом колисестве условий могут быть громоздкими. Там, где надо повторять проверку одного и тогоже значения переменной есть более элегантное решение с помощью оператора switch.

Общая форма оператора switch

     switch (проверяемая-переменная){
     case значение1: 
         код-для-высполнения-при-переменная==значение1;
         break;
     case значение2: 
         код-для-высполнения-при-переменная==значение2;
         break;
     case значение3: 
         код-для-высполнения-при-переменная==значение3;
         break;
     default: 
         код-для-выполнения-если-значение-переменной-не-предусмотрено-в-case;
     }

Важно: каждый case должен закрываться в конце с помощью break. Благодяря break программа прекращает выполнять код и выходит из условного оператора.

Задача №3

     import java.util.Scanner; // импорт сканера
 
     class Test {
	     public static void main(String args[]){
		     System.out.println("Введите 1, 2, 3 или 4");
		     Scanner scan = new Scanner(System.in);
		     int number = scan.nextInt();
 
		     switch (number){
		     case 1: System.out.println ("Вы ввели число 1"); break;
		     case 2: System.out.println ("Вы ввели число 2"); break;
		     case 3: System.out.println ("Вы ввели число 3"); break;
		     case 4: System.out.println ("Вы ввели число 4"); break;
		     default: System.out.println("Вы ввели неправильное число");
		}
	}
}

Комментарии:

Проверяем значение переменной number

switch (number)

Для этого прописываем все возможные значения переменной number и код, который будет выполняться при том или ином занчении 

	     case 1: System.out.println ("Вы ввели число 1"); break;
         case 2: System.out.println ("Вы ввели число 2"); break;
         case 3: System.out.println ("Вы ввели число 3"); break;
         case 4: System.out.println ("Вы ввели число 4"); break;

Если пользователь введет число, которое не равно 1, 2, 3 или 4 тогда отработает эта строчка:

default: System.out.println("Вы ввели неправильное число");
