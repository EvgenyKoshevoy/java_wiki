Логирование

    System.err.println - стандартный вывод записи в консоль (для быстрого дебага).

    Уровни логирования:

        - OFF: логи не пишутся, все игнорится;
        - FATAL: ошибка, после которой приложение не смоежет работать и будет остановлено;
        - ERROR: уровень ошибок, когда есть проблемы которые нудно решить. Ошибка не останавливает работу, остальные запросы могут работать некоррректно;
        - WARN: логи, которые содержат предостережения;
        - INFO: для важных действий в приложении (ожидаемые действия системы);
        - DEBUG: для отладки приложения ("method1 начал работу");
        - TRACE: менее приоритетные логи для отладки;
        - ALL: для записи всех логов системы.

    Для logback:
        - TRACE
        - DEBUG
        - INFO
        - WARN
        - ERROR

    Если включить кровень логирования INFO, то будут логироваться все уровни от INFO до FATAL.

    Запись и отправка логов

    На примере log4j.
    - для записив файл - DailyRollingFileAppender;
    - для получения данных в консоль приложения - ConsoleAppender;
    - для записи логов в базу данных - JDBCAppender;
    - для контроля передачи через TCP/IP;
    - чтобы запись логов не била по быстродействию - S\ASyncAppender.

    Узлы логирования

    Создать логгер:
        Logger logger = LoggerFactory.getLogger(MainDemo.class);

    Сделать запись в лог:
        logger.trace("some text with argument={}", argument);
        logger.debug("some text script={}", script);
        logger.info("some text port={}", port);
        logger.warn("some text");
        Logger.error("some text host={}", host);

    Пример для com.github.romankh3.logginglecture.MainDemo:

    Узлы:
        RootLogger (главный узел, принимает все логи приложения)
            com   
                com.github
                    com.github.romankh3
                        com.github.romankh3.logginglecture

    Настройка Log4j.properties

    log4j.appender.CONSOLE=org.apache.log4j.ConsoleAppender
    log4j.appender.FILE=org.apache.log4j.RollingFileAppender
