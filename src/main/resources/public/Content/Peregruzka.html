Перегрузка

Перегрузка методов

Условия:
- тип или число параметров в каждом из перегружаемых методов должны быть разными
- тип возвращаемого значения не учитывается при перегрузке

Пример:

void ovlDemo() {
    //..
}

void ovlDemo(int i) {
//...
}

int ovlDemo(int a, int b) {
    //..
}

double ovlDemo(double a, double b) {
    //..
}

Следующая попытка перегрузки скомпелируется с ошибкой, т.к. тип возвращаемого значения не учитывается при перегрузке:

void ovlDemo(int a) {
    //..
}

int ovlDemo(int a) { //компилятор неразличит два этих метода
    //..
}

Перегрузка конструкторов

Пример:

MyClass {

    MyClass() {
        //..
    }

    MyClass(int i) {
        //..
    }

    MyClass(int i, int j) {
        //..
    }

}
