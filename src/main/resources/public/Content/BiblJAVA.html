Библиотеки, классы и методы в Java

Библиотека JAVA - уже готовый и протестированный код.

В виртуальной библиотеке информация разбита по пакетам ("packages") - это своеобразный аналог полочек в книжном магазине. В каждом пакете протестированный код по какому-то отдельно взятому направлению.

Например есть пакеты:

  • java.applet
  • java.lang - основной язык пакета Java
  • java.util
  • java.io
  • java.net
  • и т.д.

Схематическое изображение библиотеки в Java:

Слева изображены разные пакеты ("packages"):

  • java.applet
  • java.lang
  • java.util
  • и т.д.

В пакете лежат классы (на примере это класс math). Класс - это готовое решение, которое мы можем использовать при написании программ.

В классе есть методы (действия), которые можно выполнить с помощью класса. Например в классе math есть методы:

  • sqrt() - извлечение квадратного корня
  • sin() - вычисление синуса угла
  • cos() - вычисление косинуса угла
  • и т.д.
