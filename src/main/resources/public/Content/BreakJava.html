Оператор Break в Java

Оператор break позволяет прерывать выполнение цикла в любой удобный для программиста момент, раньше чем цикл должен был бы завершиться.

Синтаксис

Оператор break записывается так:
break;

Оператор break для выхода из цикла

	      class Test {
         public static void main(String args[]){
             int i;
             int a;
 
             i = 1;
             a = 3;
             while(i <= 5){
                 System.out.println(i + ": цикл выполняется!");
                 if (i == a)
                     break;
                 i++;
                 }
             }
         }

Когда переменная i приобрела значение 3, цикл завершился. Так же оператор break будет работать и с дргими циклами. Для этого надо вставить строчку

if (i == a);
break;

Оператор break для выхода из цикла: вложенные циклы

Как будет работать код если циклов несколько?

	     class Test {
             public static void main(String args[]){
 
                 for (int i = 1; i <= 3; i++) {
                     System.out.println("Это первый цикл! Я выполняюсь " + i + " раз");
                     for (int a = 1; a <= 10; a++) {
                         System.out.println("    Это второй цикл! Я выполняюсь " + a + " раз");
                         if (a == 2)
                             break;
                     }
                 }
             }
         }

Тут два цикла for. Во втором цикле оператор break.

В результате выполнения кода имеем:

	 Это первый цикл! Я выполняюсь 1 раз
            Это второй цикл! Я выполняюсь 1 раз
            Это второй цикл! Я выполняюсь 2 раз
         Это первый цикл! Я выполняюсь 2 раз
            Это второй цикл! Я выполняюсь 1 раз
            Это второй цикл! Я выполняюсь 2 раз
         Это первый цикл! Я выполняюсь 3 раз
            Это второй цикл! Я выполняюсь 1 раз
            Это второй цикл! Я выполняюсь 2 раз

Первый цикл выполняется 3 раза, как и должен. Второй - 2 раза. Т.е. оператор break прерывает только тот цикл, который его непосредственно вызывает.

Оператор break и оператор switch

	     import java.util.Scanner; // импорт сканера
 
         class Test {
	         public static void main(String args[]){
		         System.out.println("Введите 1, 2, 3 или 4");
		         Scanner scan = new Scanner(System.in);
		         int number = scan.nextInt();
 
		         switch (number){
		         case 1: System.out.println ("Вы ввели число 1"); break;
		         case 2: System.out.println ("Вы ввели число 2"); break;
		         case 3: System.out.println ("Вы ввели число 3"); break;
		         case 4: System.out.println ("Вы ввели число 4"); break;
		         default: System.out.println("Вы ввели неправильное число");
		         }
	         }
         }

Каждый кейс заканчивается оператором break. Тут он как бы говорит "Ты нашел то что искал, зачем работать дальше?"

Заключение

  • break вызывается лаконично - просто break и точка с запятой;
  • break работает одинаково для всех циклов - while, for, do...while и др.;
  • break прерывает выполнение только одного цикла - того, который его непосредственно вызывает;
  • break применяется в операторе switch.
