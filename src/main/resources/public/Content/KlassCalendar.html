Класс Calendar

    Работаем с реализацией класса GregorianCalendar.

Calendar calendar = new GregorianCalendar(2017, 0, 25);    //создать календарь с датой 25 Января 2017 года

getTime() - получить из календаря простую дату (т.е. объект Date)

Пример:
Calendar calendar = new GregorianCalendar(2017, 0, 25);
Date date = calendar.getTime();

В качестве обозначений месяцев можно ипользовать константы.
Пример:
Calendar calendar = new GregorianCalendar(2017, Calendar.JANUARY, 25); //Calendar.JANUARY одна из констант для обозначения месяца

set() - позволяет установить все поля календаря по отдельности
Пример:
Calendar calendar = new GregorianCalendar();
calendar.set(Calendar.YEAR, 2017);
calendar.set(Calendar.MONTH, 0);
calendar.set(Calendar.DAY_OF_MONTH, 25);
calendar.set(Calendar.HOUR_OF_DAY, 19);
calendar.set(Calendar.MINUTE, 42);
calendar.set(Calendar.SECOND, 12);

add() - прибавление/вычитание значений
Пример:
calendar.add(Calendar.MONTH, -2); //откатит на два месяца назад с переводом года

roll() - прибавляет/убавляет значения не затрагивая другие поля
Пример:
calendar.roll(Calendar.MONTH, -2);

get() - получить поля объекта GregorianCalendar по отдельности
.get(Calendar.КОНСТАНТА);

Создание даты до нашей эры:
GregorianCalendar cannes = new GregorianCalendar(216, Calendar.AUGUST, 2);
cannes.set(Calendar.ERA, GregorianCalendar.BC);
