Java основы

    JDK - java development kit. Содержит все инструменты, исполняемые и бинарные файлы, необходимые для компиляции, отладки и выполнения программ. Является платформозависимым. Является надстройкой JRE т.к. содержит JRE с Java компилятором, отладчиком и базовыми классами.

    JVM - виртуальная машина JAVA. Преобразует байт-код в машинный код. Зависит от платформы, предоставляет основные функции, такие как управление памятью, сборка мусора и т.д.

    JRE - Java Runtime Environment. Является реализацией JVM. Предоставляет платвофрму для выполнения программ. Состоит из виртуальной машины, бинарных файлов и других классов. Не содержит инструментов для разработки (компилятор, отладчик и т.д.).

   JIT - Java-in-time compiler. Оптимизирует байт код, уменьшая общее время необходимое для компиляции байт-кода в машинный код.

    В техологии Java существует несколько основных направлений:
        - Java Enterprise Edition (JavaEE, J2EE) - технология создания ПО уровня больших корпораций.
        - Java Standart Edition (Java SE, J2SE) - технология создания пользовательских приложений.
        - Java Micro Edition (Java ME, J2ME) - создание приложений для смартфонов, КПК и других маломощных систем.
        - Java Card - технология для устройств типа смарт-карт.
