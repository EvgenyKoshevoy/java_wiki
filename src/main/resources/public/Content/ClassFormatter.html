Класс Formatter

    Конструкторы:
        Formatter()
        Formatter(Appendable буфер)
        Formatter(Appendable буфер, Locale региональные_настройки)
        Formatter(String имя_файла) throws FileNotFoundException
        Formatter(String имя_файла, String набор_символов) throws FileNotFoundException, UnsupportedEncodingException
        Formatter(File файл_вывода) throws FileNotFoundException
        Fprmatter(OutputStream поток_вывода)

    Методы:
        void close() - закрывает объект высвобождая ресурсы

        void flush() - очищает буфер отформатированных данных, все несохраненный данные выводятся по месту назначения

        Formatter format(String форматирующая_строка, Object ... аргументы) - форматирует указаннфе аргументы в соответствии со спецификациями формата, которые содержит форматирующая строка.

        Formatter format(Locale региональные_настройки, String форматирующая_строка, Object ... аргументы) - то же, с региональными параметрами

        Locale locale() - возвращает региональные настройки вызывающего объекта

        Appendable out() - возвращает ссылку на базовый объект, который служит в качестве адресата для выводимых данных

        String toString() - возвращает объект типа String содержащий отформатированные данные

    Общая форма:
        %<признак_формата><спецификатор_минимальной_ширины_поля><спецификатор_точности><спецификатор_формата>

    Спецификаторы формата:

    %а    Шестнадцатеричное значение с плавающей точкой
    %А

    %b    Логическое значение
    %В

    %c    Символ

    %d    Десятичное целое значение

    %h    Хеш-код аргумента
    %Н

    %е    Экспоненциальное представление числа
    %Е

    %f    Десятичное число с плавающей точкой

    %g    Используется спецификатор формата %е или %f в зависимости
    %G    от форматируемого значения и заданной точности

    %о    Восьмеричное целое число

    %n    Вставляет знак перевода строки

    %s    Символьная строка
    %S

    %t    Время и дата
    %Т

    %х    Шестнадцатеричное целое число
    %Х

    %%   Вставляет знак %

    Форматирование даты и времени

    Указание минимальной ширины поля   

        Пример:
            %05d - 0: дополнить нулями поле, 5 - минимальная длина поля.

    Указание точности

        Спецификатор точности применяется с %f, %e, %g, %s. Указывается после спецификатора минимальной ширины (еслит такой есть), состоит из точки и целого числа.

        Если применяется с %f или %e, то спецификатор точности определяет количество отображаемых десятичных цифр.
            %10.4f - вывод числа, шириной не меньше 10 знаков, с четырьмя цифрами после запятой.
        Если применяется с %g, то спецификатор определяет количество значащих десятичных цифр. Точность по умолчанию составляет 6 цифр после запятой.

        Применительно к строковым - задает максимальную ширину поля.
            %5.7s - вывод строки длиной минимум 5 символов, максимум 7. Если строка оказывается длиннее, то конечные символы отбрасываются.

    Признаки формата

        -    выравнивание по левому краю

        #    альтернативный формат преобразования
            Применяется с %o, %x, %e, %f. Обеспечивает наличие десятичной точки, даже в том случае, если отсутствует дробная часть числа.

        0    выводимые данные дополняются 0 вместо пробелов

        пробел    положительные числа выводятся с предшествующим пробелом

        +    Числовые значения выводятся с предшествующим знаком +

        ,    Числовые значения содержат групповые разделители
            Пример: ("%,.2f", 4356783497.34) получим: 4,356,783,497.34

        (    Отрицательные числовые значения заключаются в круглые скобки
   

    Индекс аргумента

        Позволяет указывать аргумент, к которому должен применяться конкретный спецификатор формата.
       
        Указывается после знака % и имеет форму n$.
       
        Пример:
            fmt.format("%3$d %1$d %2$d", 10, 20, 30) Вывод: 30 10 20

        Можно повторно использовать аргумент, не указывая его дважды:
            fmt.format("Число %d в шестнадцатиричном формате " + "равно %1$x", 255)    Вывод: Число 255 в шестнадцатиричной форме равно ff
       
            Либо:
                fmt.format("Число %d в шестнадцатиричном формате равно %<x", 255)
